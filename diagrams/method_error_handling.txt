ERROR HANDLING BY METHOD (ENDPOINT LEVEL)

Legend:
- 302 redirect
- 400 form validation error (render form with errors)
- 403 access denied by role
- 404 object not found
- 405 method not allowed

1) GET / (dashboard)
   Errors:
   - unauthenticated -> 302 to /login/

2) GET|POST /login/
   Errors:
   - invalid credentials -> 200 with form errors

3) GET|POST /logout/
   Errors:
   - unsupported method (e.g. PUT) -> 405

4) GET|POST /profile/
   Errors:
   - unauthenticated -> 302 /login/
   - invalid profile payload on POST -> 200 with validation errors
   - file upload issue -> 400 (form/file errors)

5) GET /certificates/
   Errors:
   - unauthenticated -> 302 /login/
   - invalid sort key -> fallback to "-event_date" (no hard error)

6) GET /certificates/<pk>/
   Errors:
   - unauthenticated -> 302 /login/
   - certificate not owned by user or absent -> 404

7) GET|POST /certificates/new/
   Errors:
   - unauthenticated -> 302 /login/
   - invalid form/file -> 200 with errors
   - OCR dependency missing -> graceful fallback, extracted_text=""

8) GET|POST /calculate/
   Errors:
   - unauthenticated -> 302 /login/
   - invalid scholarship id -> 200 with form errors

9) GET /files/
   Errors:
   - unauthenticated -> 302 /login/

10) GET /management/users/
    Errors:
    - non-admin -> 302 (user_passes_test redirect flow)

11) GET|POST /management/users/create/
    Errors:
    - non-admin -> 302
    - invalid user payload -> 200 with errors
    - duplicate username -> 200 with form errors

12) GET|POST /management/users/<pk>/edit/
    Errors:
    - non-admin -> 302
    - user not found -> 404
    - invalid user/profile payload -> 200 with errors

13) GET|POST /management/users/<pk>/delete/
    Errors:
    - non-admin -> 302
    - user not found -> 404
    - wrong method other than GET/POST -> 405 (Django default view dispatch)

14) GET|POST /management/users/<pk>/adjust/
    Errors:
    - non-admin -> 302
    - user not found -> 404
    - invalid adjustment payload -> 200 with errors

15) GET /management/certificates/
    Errors:
    - non-admin -> 302

16) GET|POST /management/certificates/<pk>/edit/
    Errors:
    - non-admin -> 302
    - certificate not found -> 404
    - invalid form payload -> 200 with errors

17) GET|POST /management/certificates/<pk>/delete/
    Errors:
    - non-admin -> 302
    - certificate not found -> 404

18) GET /management/statistics/
    Errors:
    - non-admin -> 302
