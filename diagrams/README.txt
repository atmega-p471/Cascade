DIAGRAMS PACKAGE (PGAS PLATFORM)

Содержимое:
1) pgas_process.bpmn
   BPMN 2.0 XML, совместимо с bpmn.io.

2) use_cases.puml
   Use Case диаграмма в PlantUML.

3) uml_class.puml
   UML Class диаграмма (доменная модель + связи) в PlantUML.

4) use_cases.drawio.xml
   XML для diagrams.net / draw.io.

5) er.dbml
   ER-модель в формате dbdiagram.io (DBML).

6) user_stories.txt
   User Stories и критерии приемки.

7) method_error_handling.txt
   Обработка ошибок по каждому endpoint-методу.

8) method_mappings.txt
   Маппинги (роуты, формы, константы, статусы, фильтры) по методам.

9) method_io_messages.txt
   Входящие и исходящие сообщения по каждому endpoint-методу.
